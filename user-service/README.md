# **User Service Documentation**

## **1\. Обзор**

**User Service** — это микросервис, отвечающий за управление пользователями в системе (регистрация, профили, аутентификация и хранение персональных данных).

## **2\. Стек технологий**

* **Java 25**  
* **Spring Boot 4.0.6**  
* **Spring Security** (JWT аутентификация)  
* **Spring Data JPA**  
* **PostgreSQL 17**  


## **3\. Архитектура и компоненты**

Сервис изолирует логику пользователей от остальной видео-инфраструктуры.

## **4\. Безопасность и API**

Сервис защищен с помощью Spring Security.

* **Базовая аутентификация:** Используется для служебных запросов.  
* **JWT:** Основной механизм передачи токенов доступа между микросервисами.  

## **5\. Конфигурация (application.yaml)**

Ключевые параметры для настройки сервиса:

| Переменная                          | Описание                               |
|:------------------------------------|:---------------------------------------|
| SPRING\_DATASOURCE\_URL             | JDBC URL к базе данных PostgreSQL      |
| SECURITY\_JWT\_SECRET               | Секретный ключ для подписи JWT токенов |
| SPRING\_SECURITY\_USER\_NAME        | Логин для базы данных postgres         |
| SPRING_DATASOURCE_PASSWORD          | Пароль для базы данных postgres        |


## **6\. Запуск проекта**

### **Локально**

1. Убедись, что запущена БД.  
2. Сборка: 
```Bash
./gradlew clean build
```
3. Запуск: 
```Bash 
./gradlew bootRun
```

### **Docker**

Запуск через Docker Compose:

```Bash  
docker-compose up \-d user-service
```
